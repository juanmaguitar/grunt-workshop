Grunt is a (Javascript) Task Runner

Grunt is a task-based command line build tool for JavaScript projects.

Clear Ideas

  • Grunt is a command line command that allow us to execute tasks written in Javascript
  • With Grunt we can AUTOMATE A LOT: minify, concatenate, uglify, compile (sass, coffeescript), deploy...
  • Started in 2012 by Ben Allman from Boucup (Boston)
  • First commit to the project: Sep 2011

Code

module.exports = function(grunt) {
    grunt.initConfig({
        concat: {
            main: {
                src: 'src/*.js',
                dest: 'build/scripts.js'
            }
        },
        min: {
            main: {
                src: '<config:concat.main.dest>',
                dest: 'build/scripts.min.js'
            }
        }
    });
    grunt.registerTask('default', 'concat min');
};

Code example

concat: {
    main: {
        text: 'Hello world'
    }
}
