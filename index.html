Grunt is a Task Runner

Grunt is a Build Tool

Grunt is a (Javascript) Tasks Runner. Some of these tasks Build stuff (transcompiling, deploy...)

Originally described as:

Grunt is a task-based command line build tool for JavaScript projects.

Clear Ideas

  • Grunt and Grunt-plugins are installed and managed via npm, the Node.js package manager.
  • To use Grunt from comand line we have to install (globally) the Grunt's CLI → npm install -g grunt-cli
  • Inside the project we install Grunt and Grunt-plugins as node modules → npm install --save-dev grunt
  • To install dependencies of an exisitng project we use → npm install

Clear Ideas

  • The grunt command line (globally) runs the grunt package at node_modules (locally)
  • The tasks are coded (loaded and configured) in Javascript in the file Gruntfile.js
  • The "plugins" used in the Gruntfile.js are set as project-dependencies in the file package.json → npm install

Clear Ideas

  • Productivity → with Grunt we can automate everything in the client side: minify, concatenate, uglify, transcompile, deploy...
  • Community → more than 11000 grunt plugins available at NPM
  • Transcompilation → Ease the source-to-source compilation (Haml, Jade, Sass, LESS, Stylus, CoffeeScript, Dart, TypeScript, and more.)

Clear Ideas

  • Released in March 2012 by Ben Allman

Clear Ideas

  • npm install -g grunt-cli → to install grunt "command-line"
  • npm install → (inside the project) to install dependencies
  • npm install --save-dev grunt-contrib-jasmine → to install a new "plugin" and add it automatically to package.json

Using Grunt

$ git --version
$ node -v
$ npm -v
$ git clone https://github.com/juanmaguitar/grunt-workshop.git
$ cd grunt-workshop
$ npm install
$ grunt --version
$ grunt -h
$ grunt tasks
$ grunt compass
$ grunt shower
$ grunt serve

First tasks w/ Grunt (2)

$ vi Gruntfile.js
    module.exports = function(grunt) {
      grunt.loadNpmTasks('grunt-contrib-jshint');
      grunt.initConfig({
        jshint: {
          options: {
            curly: true,
            eqeqeq: true
          },
          target1: ['Gruntfile.js', 'src/**/*.js']
        }
      });
      grunt.registerTask('default', ['jshint']);
    };
$ grunt
